Tools

Tableau

Tableau is a visualization software that I use to visualize data after it has been aggregated in either RStudio or Big Query. I can create interactive dashboards that are easy to share.

Google Big Query

Google BigQuery is a cloud-based data warehouse for data analytics processing of very large read-only data sets using standard SQL.

Microsoft Excel

Microsoft Excel is a spreadsheet software that I also use to clean, analyze, and visualize data. I mostly use this for smaller sets of data less than a million rows long.
